## Data Attributes

You can use below `data-*` attributes to style the Angular Material Button component. Make sure to use below with `matButton` directive.

### data-size

The size of the button.

#### Default Value

`s`

#### Possible Values

`xs`, `s`, `m`, `l`, `xl`

### data-shape

The shape of the button.

#### Default Value

`round`

#### Possible Values

`round`, `square`
