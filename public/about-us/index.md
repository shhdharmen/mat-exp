**Mat Expressive** is an Angular component library that brings Google's Material Expressive design language to Angular applications — expressive motion, fluid shapes, and adaptive sizing built on top of Angular Material.

## The Library

`@ngm-dev/mat-expressive` is built with:

- **Angular 21** — standalone components, signals, and native control flow throughout
- **Angular Material**
- **GSAP 3** — for expressive, physics-based motion
- **PolyForm Noncommercial 1.0.0** — free for non-commercial use
