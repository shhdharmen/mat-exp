## v0.1.0

_Initial public release_

### Components

- **Button** (`matExpressiveButton`) — size, shape, toggle, and appearance variants; full GSAP motion support
- **Icon Button** — icon-only variant with the same motion system
- **Button Group** — grouped button layout with shared size/shape context
- **Split Button** — primary action + overflow menu in a single expressive control
- **FAB Menu** — floating action button with an animated overflow menu
- **Loading Indicator** — animated progress indicator with expressive motion

### Docs Site

- SSG documentation site with markdown-based content
- API reference with auto-generated manifest from TypeScript compiler API
- Live playground per component
- Search (Pagefind)
- Subdomain versioning (`v1.expressive.angular-material.dev`, …)
